# команды с файлами
rm / rm -rf - удаление файла / директории
cd - перейти в директорию
nano / vim - редактировать файл в терминале
touch / mkdif - создать файл / директорию
chmod - изменить права

# Навигация с файлами
find - поиск по названию, весу и типу файл/а (медленный)
locate - поиск только по названию (быстрый)
tree - для документации, аудита структуры папок/файлов (строит дерево файлов)
ln - создание символьных ссылок (при версионности или разных версиях языка к примеру)
stat - информация о файле (кто когда изменил, проверить права доступа)
file - определить тип файла
